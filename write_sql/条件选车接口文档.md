### 条件选车接口文档
- 请求地址
```
   /v5/sms/car/specs
```

- 请求方式:POST
- 请求参数
```
n                   int类型,页数

brands_name         string类型,车品牌名称

price               string类型,价格
                    1表示8万以下,2表示8~15万,3表示15~20万,4表示20~30万,5表示30~50万,
                    6表示50~70万,7表示70~100万,8表示100万以上
                    当选择多个的时候用逗号分隔,例如:
                        "1,3,7",下同

grade               string类型,级别
                    1代表微型车,2代表小型车,3代表紧凑型车,4代表中型车,5代表中大型车,
                    6代表大型车,7代表跑车,8代表SUV,
                    9代表MPV,10代表微面,11代表微卡,12代表轻客,13代表皮卡

disp                string类型,排量
                    1代表排量在1L以下,2代表排量在1.1-1.6L,3代表排量在1.7-2.0L,4代表排量2.1-2.5L,5代表排量在2.6-3.0L,
                    6代表排量在3.1-4.0L,6代表排量在4.0L以上

seat_num            string类型,座位数
                    1代表2座,2代表4座,3代表5座,4代表6座,5代表7座,6代表7座以上

struct              string类型,排量
                    1表示两厢,2代表三厢,3代表掀背,4代表旅行版,5代表硬顶敞篷跑车,6代表软顶敞篷跑车,7代表硬顶跑车,8代表客车,9代表货车

energy              string类型,能源
                    1代表汽油,2代表柴油,3代表油电混合,4代表纯电动,5代表插电式混动,6代表增程式

gbox                string类型,变速箱
                    1代表手动,2代表自动

config              string类型,配置
                    1代表天窗,2代表电动调节座椅,3代表GPS导航,4代表车型ESP,5代表氙气大灯,6代表真皮座椅,7代表定速巡航,8代表全自动空调,9代表倒车影像,10代表无钥匙启动
                    11代表座椅加热,12代表自动泊车

intake_mode         string类型,配置
                    1表示自然吸力,2表示涡轮增压,3表示机械增压

drive_mode          string类型,驱动方式
                    1表示前驱,2表示后驱,3表示四驱


注:
                    以上配置参数当有多个的时候都是以逗号拼接作为字符串,传给后台

```
